# arch-i3wm-x11

Stable and minimal i3wm (X11) setup for low-end Intel laptops.

## Features
- i3wm (X11)
- Polybar + Rofi
- Picom (auto-disable fullscreen)
- xsecurelock (modern lockscreen)
- Volume & brightness OSD
- Catppuccin & Nord theme switch

## Install
```bash
git clone https://github.com/adrenaline404/arch-i3wm-x11
cd arch-i3wm-x11
chmod +x install.sh
./install.sh
startx
